# Mobile App

## What is the Bastion mobile app?

The mobile app is a client for iOS and Android that connects to **your** Bastion — the one running on your computer or server. It stores no data: it's just an interface to chat with your agent from anywhere.

The model is similar to apps like Jellyfin or Nextcloud: you download the app, point it to your instance, and you're done.

---

## How to connect

The connection is made through a temporary code generated by Bastion itself. You don't need to configure anything manually in the app.

### Step 1 — Generate the connection code

In Telegram or WhatsApp, send:

```
/connect-app
```

Bastion will respond with a code in the format `BAST-XXXX-XXXX` and a QR code. This code is valid for **5 minutes** and can only be used **once**.

### Step 2 — Open the app and paste the code

In the mobile app, on the setup screen, paste the `BAST-XXXX-XXXX` code or scan the QR code.

### Step 3 — Done

The app connects to your Bastion and generates an access token that is securely stored on your phone. You don't need to repeat this process — the token lasts 90 days.

---

## Managing connected devices

To see which devices are connected to your Bastion:

```
/devices
```

Bastion lists all devices with name and status.

To disconnect a specific device:

```
/revoke my-iphone
```

Access is revoked immediately. If someone has the app open on that device, the next request will be rejected.

---

## Security

- The connection code expires in 5 minutes and only works once — even if unused, it cannot be reused after expiry
- The access token is stored in the keychain (iOS) or keystore (Android), never in insecure storage
- If you lose your phone, use `/revoke` via Telegram or WhatsApp to revoke access immediately
- The app never stores your conversations — everything stays in your Bastion

---

## Where to download the app

The app is available on the App Store (iOS) and Google Play (Android). Search for "Bastion AI" or visit [bastion.ai/app](https://bastion.ai/app).
