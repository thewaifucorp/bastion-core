# Connecting the Mobile App

The Bastion mobile app is a client for iOS and Android that connects directly to **your** instance — whether running on your computer or a VPS. It stores no data: it's just an interface to chat with your agent from anywhere.

---

## Before you start

For the app to connect, Bastion needs to be accessible from the internet. This means:

- **Running on a VPS with a configured domain** — recommended. See [vps-setup.md](vps-setup.md).
- **Running locally with a tunnel** — use [ngrok](https://ngrok.com) or [Cloudflare Tunnel](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/) to temporarily expose your instance.

If Bastion is only accessible on your local network (home Wi-Fi), the app will only work when you're connected to that network.

---

## How to connect

### Step 1 — Generate the connection code

In Telegram or WhatsApp, send:

```
/connect-app
```

Bastion responds with:
- A code in the format `BAST-XXXX-XXXX`
- A QR code to scan directly from the app

The code is valid for **5 minutes** and can only be used **once**.

### Step 2 — Open the app

Download the app from the [App Store](https://apps.apple.com) or [Google Play](https://play.google.com) — search for "Bastion AI".

On the home screen, tap "Connect to my Bastion".

### Step 3 — Paste the code or scan the QR

You have two options:
- **Type the code**: paste the `BAST-XXXX-XXXX` in the indicated field
- **Scan the QR**: tap the camera icon and point it at the QR code

### Step 4 — Enter your Bastion address

The app will ask for your instance address. Use the domain you configured:

```
https://bastion.yourdomain.com
```

If you're using ngrok locally, use the address generated by ngrok.

### Step 5 — Done

The app connects, validates the code, and saves an access token in the keychain (iOS) or keystore (Android). You don't need to repeat this process — the token lasts **90 days**.

## Managing connected devices

To see all devices with access to your Bastion:

```
/devices
```

Example response:
```
Connected devices:
• my-iphone — connected 3 days ago
• work-ipad — connected 12 days ago
```

To revoke access for a device:

```
/revoke my-iphone
```

Access is revoked immediately. If the app is open on that device, the next message will be rejected.

---

## Security

- The `BAST-XXXX-XXXX` code expires in 5 minutes and only works once — even if unused, it cannot be reused after expiry
- The access token is stored in the keychain (iOS) or keystore (Android), never in insecure storage
- If you lose your phone, use `/revoke` via Telegram or WhatsApp to revoke access immediately
- The app never stores your conversations — everything stays in your Bastion

---

## Common issues

**"Invalid or expired code"**
The code lasts 5 minutes. Generate a new one with `/connect-app` and try again.

**"Could not connect to server"**
Check that the address is correct and Bastion is accessible. Test in your browser: `https://bastion.yourdomain.com` should respond.

**App connected but not responding**
Check that the TOTP session is active. The mobile app also requires TOTP authentication in new sessions — the app will request the code automatically.
